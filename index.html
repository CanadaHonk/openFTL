openFTL

openFTL is an extremely WIP open recreation of FTL (1 day old) not a decompile/etc, it is a recreation from scratch. (kinda like OpenTTD/etc) you need an ftl.dat file to try (it is where your FTL binary is), openFTL loads all of FTLs resources from it. implemented: - reading .dat files - reading .font files - custom text rendering using .font's - title screen - options window next up? - tooltips - hangar/customize screen
